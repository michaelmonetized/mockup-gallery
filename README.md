# Web Design Portfolio — HurleyUS

Live showcase of 10 industry-specific web design mockups. Modern, responsive, conversion-focused.

## Quick Start

```bash
npm install
npm run dev
# Open http://localhost:3000
```

## Deployment

```bash
git add .
git commit -m "Deploy web design mockups"
git push origin main
# Then visit vercel.com → Import → Deploy
```

## Industries Included

## Design Philosophy

- ✅ Modern typography & hierarchy
- ✅ High contrast for readability
- ✅ Mobile-first responsive design
- ✅ Clear CTAs and conversion focus
- ✅ Trust signals (testimonials, ratings, social proof)
- ✅ No generic AI — real design polish
